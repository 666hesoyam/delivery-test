  • Віджети

Логістична компанія «Делівері» спільно з «BGS Solution» - офіційним партнером компанії «1С» в Україні - розробили модуль доставки «Делівері» для 1С. Даний програмний продукт покликаний цілком задовольнити потреби вимогливих клієнтів в якісній логістиці. Модуль доставки «Делівері» відповідає всім стандартам технологічної платформи і підтримує інтеграцію з більшістю типових конфігурацій програмних продуктів фірми «1С» версії 8.2.

    Модуль корисний

  • •Підприємцям, які ведуть свою діяльність в «1С: 8.2» та здійснюють перевезення вантажу через логістичну компанію «Делівері».

  • •Підприємцям, які ведуть свою діяльність в «1С: 8.2» і тільки шукають в обличчі «Делівері» надійного партнера по вантажоперевезенням.

Завантажити модуль

Ключові можливості модуля

  • Оперативність

    Швидке оформлення замовлення на відправку вантажу з зазначеного місця.

  • Автономність

    Пошук та друк уже створених квитанцій та стікерів на перевезення.

  • Контроль

    Цілодобовий контроль за перевезеннями завдяки інформації про місцезнаходження вантажу та статусу його оплати1

  • Свобода вибору

    Виписка нових квитанцій двома способами: самостійно, або на основі документу «Замовлення покупця».

  • Точність

    Попередній прорахунок вартості та строків доставки

  • Зручність

    Отримання даних із квитанцій за номером документу в 1С.

  • Переваги 1С

  • Відео інструкція по 1С

Принципи роботи

  • Крок 1

    Модуль доставки «Делівері» є безкоштовним, запускається як зовнішній файл та не вимагає залучення програмістів ззовні.

  • Крок 2

    Для підключення модулю варто зазначити API ключ та секретний API ключ, які клієнт отримує після запиту в Особистому кабінеті на сайті компанії, або за
    запитом до відділу IT.

  • Крок 3

    За допомогою Модулю доставки «Делівері» облікова система 1 С звертається до серверу «Делівері».

  • Крок 4

    Клієнт щоразу отримує актуальну інформацію щодо складів відправлення та отримання, включно з адресами, телефонами та графіками роботи.

Для обраних представництв визначається можливість доставки/ забору вантажу, категорій відправлення, додаткових послуг, а також необхідність оформлення документів ЗЕД.

Під час оформлення квитанції на перевезення можливо додавати нових отримувачів та адреси доставок/ заборів, або обирати їх із раніше створених квитанцій

Гарантії безпеки та вигоди від використання

Модуль доставки «Делівері» 1С використовує API ключ та секретний API ключ, за допомогою яких відбувається шифрування даних. Це забезпечує цілковиту секретність передачі інформації.

На сьогодні програмні продукти 1 С цілком заслужено вважаються стандартами програм, які забезпечують коректну роботу бухгалтерського та управлінського обліку більшості підприємств України та країн колишнього СНД.

Завдяки Модулю доставки «Делівері» клієнти компанії отримують зручну взаємодію власної системи 1С із сервером логістичного оператора через API.

Модуль дозволяє автоматично заповнювати основну частину даних із бази 1С при оформленні квитанції на перевезення – це істотно полегшує і прискорює оформлення документації.
